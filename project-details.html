Email-to-Sticky Notes Tool

This custom Python script streamlines internal communication by converting Outlook emails into color-coded sticky notes on Windows desktops. It's designed for use in fast-paced office environments to quickly capture tasks or reminders directly from email content.

  • Languages: Python, Powershell
  • Libraries: pywin32, os, subprocess
  • Tools: Outlook API, Sticky Notes app integration
  • Features: Auto-extracts subject, body; auto-formats note content; supports batch conversion
  • Challenge: Interfacing with Windows Sticky Notes required creative workarounds using Powershell

Education App for Kenya

This conceptual mobile app is aimed at bridging educational gaps in rural Kenya. It focuses on delivering lessons, quizzes, and teacher communication in areas with limited internet access.

  • Platform: Android (concept only)
  • Languages: Java (Android), Figma for design
  • Features: Offline lesson caching, SMS-based assignment submission, lightweight UI
  • Challenge: Designing for extremely low bandwidth and power usage environments
